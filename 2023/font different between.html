font question in different browser

Tue 10 October 2023 Tue 10 October 2023
By DrinDr
Category: notes
Tags: font web

While i try to configure the theme of this static web, i found that the font "monospace" is totally different in firefox and edge.

At first, i noticed that the monospace font is different between them. so i import the font from googleapi to ensure the web use the same font in different browser.

However, it did not being repaired perfectly. The apearance of the same font is different as well.

In fact, there is user agent style for browser to render a web defaultly. agent style and Cascade mechanism in different browser are varied. When i add body{font-weight: 400;}, the <h2> tag in firefox can successfully inherit automatically, but in edge it will consist on its agent style due to the different mechanism.

in the end, i use the * to select all the attributes in stylesheet which is a bit hacky. anyway, it works nearly perfect regardless a little different still remain.
